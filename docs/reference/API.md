# API

Canonical endpoint inventory for the API Reference section of the documentation page.

## Accounts & Snapshots Endpoints

| Method | Path | Handler | Purpose |
| --- | --- | --- | --- |
| GET | /api/accounts | api_accounts | List account summaries including equity, benchmark, PnL deltas, and latest snapshot timing. |
| GET | /api/accounts/compare | api_accounts_compare | Return all accounts with live equity stats and latest backtest metrics for side-by-side comparison. |
| GET | /api/accounts/{account_name} | api_account_detail | Get one account detail, latest backtest summary, recent snapshots, and recent trades. |
| POST | /api/actions/snapshot-all | api_snapshot_all | Create equity snapshots for all managed accounts. |
| POST | /api/actions/snapshot/{account_name} | api_snapshot | Create an equity snapshot for one account. |
| GET | /health | health | Simple health check. Returns {"status":"ok"} when backend is up. |

## Admin Endpoints

| Method | Path | Handler | Purpose |
| --- | --- | --- | --- |
| POST | /api/admin/accounts/create | api_admin_create_account | Create a new account with core fields, risk policy, options settings, and rotation settings. |
| POST | /api/admin/accounts/delete | api_admin_delete_account | Delete an account and its dependent data after explicit confirmation. |
| GET | /api/admin/exports/csv | api_csv_exports | List available CSV exports generated by repository data operations. |
| GET | /api/admin/exports/csv/preview | api_csv_export_preview | Preview rows from a selected CSV export file with a configurable row limit. |

## Logs Endpoints

| Method | Path | Handler | Purpose |
| --- | --- | --- | --- |
| GET | /api/logs/files | api_log_files | Return available .log files from the configured logs directory. |
| GET | /api/logs/{file_name} | api_log_file | Return recent lines from a log file with optional filtering. |

## Backtesting Endpoints

| Method | Path | Handler | Purpose |
| --- | --- | --- | --- |
| GET | /api/backtests/latest/{account_name} | api_latest_backtest_for_account | Get the latest persisted backtest run summary for one account. |
| POST | /api/backtests/preflight | api_backtest_preflight | Preview warnings for a proposed backtest configuration without executing it. |
| POST | /api/backtests/run | api_run_backtest | Run a single backtest window and persist the result payload. |
| GET | /api/backtests/runs | api_backtest_runs | List recent backtest runs across accounts. |
| GET | /api/backtests/runs/{run_id} | api_backtest_run_report | Get the full backtest report payload for a single run id. |
| POST | /api/backtests/walk-forward | api_run_walk_forward | Execute rolling walk-forward windows and return aggregate summary metrics. |
